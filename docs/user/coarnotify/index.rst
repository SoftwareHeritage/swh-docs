Software Heritage COAR Notify Inbox
===================================

.. admonition:: Intended audience
   :class: important

   Partners working to connect our COAR Notify service to their IT systems.

The COAR Notify service allows a partner to submit metadata to the |swh| archive.

It is currently experimental and **not** suited for production.
